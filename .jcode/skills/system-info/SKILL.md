# System Info Skill

## Descripción
Proporciona información completa del entorno de Termux/Android actual.

## Uso
Útil para obtener un resumen rápido del sistema.

## Comandos Disponibles
- Estado de memoria y CPU
- Espacio en disco
- Herramientas instaladas
- Versiones de lenguajes
