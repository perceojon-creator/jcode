# Code Helper Skill

## Descripción
Ayuda con desarrollo de código en múltiples lenguajes.

## Capacidades
- Escribir código nuevo
- Revisar y optimizar código existente
- Debugging
- Crear tests
- Documentación

## Lenguajes soportados
- Python, JavaScript/TypeScript, Go, Rust, C/C++
- Bash scripts
- SQL (PostgreSQL, SQLite)
- JSON, YAML, Markdown
