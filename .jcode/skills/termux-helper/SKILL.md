# Termux Helper Skill

## Descripción
Ayuda con la gestión de paquetes y herramientas en Termux.

- Instalar nuevos paquetes
- Actualizar sistema
- Verificar dependencias

- golang, rust, cmake, ninja
- sqlite, postgresql, redis
- vim, nano, zsh, htop, tmux
